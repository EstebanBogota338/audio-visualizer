# Guía del Equipo

## Workflow

1. Hacer pull de main
2. Entrar a tu branch
3. Programar
4. Commit claro
5. Push

---

## Reglas

- No trabajar en main
- No modificar carpetas ajenas
- Mantener código limpio
- Avisar antes de restructurar
