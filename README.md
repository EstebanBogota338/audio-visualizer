# Audio Visualizer

Visualizador de audio web en tiempo real usando FFT y Canvas API.

---

## Tecnologías

- HTML
- CSS
- JavaScript
- Web Audio API
- Canvas API

---

## Features

- Micrófono en vivo
- FFT visualizer
- UI cyberpunk
- Render en tiempo real

---

## Estructura

audio/ → procesamiento de audio  
visual/ → render visual  
ui/ → controles y eventos  
utils/ → utilidades  

---
